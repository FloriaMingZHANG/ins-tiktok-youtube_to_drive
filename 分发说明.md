# 如何打包发给别人用（Windows / Mac 通用）

本程序**同时支持 Windows 和 Mac**，对方只需有 Python 环境并按说明做一次性配置即可运行。

---

## 一、你打包时要做的事

### 1. 打成 zip 时**务必排除**以下内容（涉及隐私或体积大）

- **不要打包进去**：
  - `.venv` 文件夹（虚拟环境，体积大且和本机 Python 绑定）
  - `.env`（你自己的表格地址、文件夹 ID 等）
  - `credentials.json`（你的服务账号密钥）
  - `client_secret.json`（你的 OAuth 密钥）
  - `token.json`（你的登录 token）
- **要保留在 zip 里**：
  - `main.py`
  - `requirements.txt`
  - `config.example.env`
  - `run.bat`（Windows 双击运行）
  - `run.sh`（Mac 运行）
  - `README.md`、`零基础操作指南.md`、`上传到我的网盘-OAuth配置.md`、`分发说明.md`（本文件）

### 2. 打包方式示例

- **Mac**：在「访达」中选中 `ins_to_drive` 文件夹内上述要保留的文件和文档（不要选 `.venv`、`.env`、`credentials.json` 等），右键 →「压缩」。
- **Windows**：进入 `ins_to_drive`，Ctrl 选中上述文件/文件夹（同样排除 .venv、.env、*.json 等），右键 →「发送到」→「压缩(zipped)文件夹」。

或命令行（在 `ins_to_drive` 的**上一级**目录执行，且当前目录没有 .env/credentials 等敏感文件时）：

```bash
# 示例：只打包需要的（根据你实际路径改）
zip -r ins_to_drive.zip ins_to_drive -x "ins_to_drive/.venv/*" "ins_to_drive/.env" "ins_to_drive/credentials.json" "ins_to_drive/client_secret.json" "ins_to_drive/token.json"
```

## 二、对方收到 zip 后要做的事（Windows / Mac 通用）

1. **安装 Python**（若未安装）  
   - 从 https://www.python.org/downloads/ 下载安装。  
   - Windows 安装时务必勾选「Add Python to PATH」。

2. **解压**  
   - 把 zip 解压到任意目录（如「桌面」或「文档」），得到 `ins_to_drive` 文件夹。

3. **第一次使用：安装依赖**  
   - **Windows**：在 `ins_to_drive` 文件夹里按住 Shift + 右键 →「在此处打开 PowerShell 窗口」（或「命令提示符」），依次执行：
     ```bat
     python -m venv .venv
     .venv\Scripts\activate
     pip install -r requirements.txt
     ```
   - **Mac**：打开「终端」，执行：
     ```bash
     cd 把 ins_to_drive 的路径拖到这里
     python3 -m venv .venv
     source .venv/bin/activate
     pip install -r requirements.txt
     ```

4. **按说明配置 Google**  
   - 让对方打开 **《零基础操作指南.md》**，从「第四步：在 Google 云端创建服务账号」做到「第八步：运行脚本」；  
   - 若需要上传到「我的云端硬盘」，再按 **《上传到我的网盘-OAuth配置.md》** 配置 OAuth。

5. **以后每次运行**  
   - **Windows**：双击 **`run.bat`**，或在已打开命令行的该目录下执行 `run.bat`。  
   - **Mac**：在终端里 `cd` 到 `ins_to_drive` 后执行 `bash run.sh`；或先给 run.sh 执行权限（`chmod +x run.sh`），再 `./run.sh`。

---

## 三、总结

| 项目       | 说明 |
|------------|------|
| 系统       | Windows、Mac 均可，同一份 zip。 |
| 前提       | 对方电脑需安装 Python（3.9 或以上建议 3.10+）。 |
| 第一次     | 解压 → 建虚拟环境 + 安装依赖 → 配置 .env 与 Google 凭证。 |
| 以后运行   | Windows 双击 `run.bat`；Mac 终端运行 `bash run.sh` 或 `./run.sh`。 |
| 不要打包   | .venv、.env、credentials.json、client_secret.json、token.json。 |

对方按《零基础操作指南》和《上传到我的网盘-OAuth配置》做完一次配置后，即可在各自系统上正常使用。
